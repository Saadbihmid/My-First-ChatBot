# Alfred - AI Chatbot

A conversational AI chatbot built with Python and Google's Gemini API.
Unlike his previous amnesiac version, Alfred now remembers everything. 😄

## Features
- 💬 Conversational memory (remembers the full conversation across sessions).
- ⌨️ Streaming responses (types letter by letter like a real assistant).
- 🎩 Custom personality via system prompt.

## Technologies Used
- Python.
- Google Gemini API.
- google-genai library.

## How to Run
1. Clone the repository.
2. Install dependencies: `pip install google-genai`.
3. Set your 'Gemini_api' as an environment variable.
4. Run: `py Alfred.py`.
